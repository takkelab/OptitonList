# PWA化に必要なコマンド集

## 1. 依存パッケージのインストール

```bash
npm install vite-plugin-pwa -D
```

または

```bash
yarn add vite-plugin-pwa -D
```

## 2. ビルドコマンド

```bash
# 本番ビルド
npm run build

# ローカルプレビュー
npm run preview
```

## 3. アイコン一括生成（オプション）

元画像（1024×1024推奨）を用意してから：

```bash
# pwa-asset-generatorをインストール
npx pwa-asset-generator [元画像.png] public --icon-only

# または具体的なサイズ指定
npx pwa-asset-generator logo.png public --icon-only --opaque false --background "linear-gradient(to right, #4facfe 0%, #00f2fe 100%)"
```

## 4. Vercelデプロイ

### 初回デプロイ
```bash
# Vercel CLIをインストール（グローバル）
npm install -g vercel

# デプロイ
vercel

# 本番環境にデプロイ
vercel --prod
```

### GitHubと連携する場合
1. GitHubにプッシュ
```bash
git add .
git commit -m "PWA対応完了"
git push origin main
```

2. Vercelで自動デプロイ設定
- https://vercel.com でGitHubリポジトリを連携
- 以降、mainブランチへのプッシュで自動デプロイ

## 5. 開発サーバー起動

```bash
npm run dev
```

## 6. 型チェック（TypeScript使用時）

```bash
npx tsc --noEmit
```

## 7. 依存関係の確認

```bash
npm list vite-plugin-pwa
```

## トラブルシューティング用コマンド

### キャッシュクリア
```bash
rm -rf node_modules
rm package-lock.json
npm install
```

### ビルドキャッシュクリア
```bash
rm -rf dist
rm -rf .vite
npm run build
```
